# Note Taker

## Description

The user provided a story stating:

AS A small business owner

I WANT to be able to write and save notes

SO THAT I can organize my thoughts and keep track of tasks I need to complete

I learned the use of the express node module, and gained further skill in routing servers/

## Installation

The back end developer will need to add express version 4.18.2 via npm install.

npm start will activate the server on Port 3001
